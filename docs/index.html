Nosso Jogo:

    O grande mestre Toshi passeava no vale quando de repente... PUM! Sua coxinha começa a voar.
    Sendo o grande mestre que ele é, Toshi resolve ir a busca de sua preciosa.
    A aventura começa agora!

Como jogar:

    Voce, jogador, irá controlar o personagem com as teclas 'A' ,'D' e 'Espaço'.
    A = Esquerda , D = Direita e
    Espaço = Pula.
    Lembre de ser rápido, coxinhas estragam...
Dentro do jogo é possivel ver mais informações ;)

Download

Atenção!

    Para conseguir jogar é preciso ter instalado o VSCode e o pygame em seu computador,
    se ja possui os programas faça o download, caso contrário siga os passos logo abaixo.

Instalação:

  1. Entre no site do VSCode e baixe a versão de acordo com seu sistema operacional(Windows,macOS,Linux).
  2. Em seguida pressione as teclas "Windows + R" (COMMAND + R no macOS) e digite "cmd" para abrir seu prompt de comando
  3. Digite "pip install pygame" no seu prompt de comando para instalar a extensão do pygame
  4. Agora pronto! Basta abrir o arquivo zip no seu VSCode e se divertir!
